Time Sheet

Document your breaks and time you work.

Export your document at the end of the day.

Log your times on the fly!

The application allows you to log you next event starting with clocking in. This will alternate as there is no need to clock in two times in a row.

Edit your logs!

The application allows you to edit your clock events. This will update preceding or succeeding events depending on overlap.

Save your events to a JSON formatted text file!

The application allows you to save your events to a file on you local drive.

Load your events from a JSON formatted text file!

The application allows you to load the events from a file from your disk such that you can review, add, edit, and save your file again.
